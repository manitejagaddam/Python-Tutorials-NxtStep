# Day 3 - Python Learning Guide

## Topics Covered:

This directory contains organized Python learning materials for Day 3:

### 1. **01_type_conversion_revision.py**
   - Quick revision of type conversion
   - Converting strings to integers
   - Understanding data types with type()

### 2. **02_conditional_statements.py**
   - If-else structure review
   - Arithmetic operators (+, -, *, /)
   - Comparison operators (<, >, <=, >=, !=, ==)

### 3. **03_elif_statements.py**
   - elif (else if) control flow
   - Grade system homework problem
   - Proper use of elif vs multiple if statements
   - How elif stops checking after first match

### 4. **04_strings.py**
   - Strings as stream of characters
   - String length using len()
   - String concatenation using +
   - String multiplication using *

### 5. **05_loops_and_range.py**
   - For loops basics
   - range() function with start, end, step
   - Loop examples and patterns
   - Homework challenges: print even/odd numbers

## Homework Challenges:
   1. Print even numbers till 100 (0, 2, 4, 6, ...)
   2. Print odd numbers till user input (1, 3, 5, 7, ...)
