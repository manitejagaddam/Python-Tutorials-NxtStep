1. Changing the list items 

```bash
    nums = [10, 20, 30, 40]
    nums[1] = 10
    print(nums)

    [10, 10, 30, 40]
```


2. List replacing

```bash
    nums = [10, 20, 30, 40]
    nums[1] = 10
    print(nums)

    [10, 10, 30, 40]
```

3. Append method

```bash
    nums = [10, 20, 30, 40]
    nums.append(3=50)
    print(nums)

    [10, 20, 30, 40, 50]
```

4. insert method

```bash
    nums = [10, 20, 30, 40]
    nums.insert(1, 50)
    print(nums)

    [10, 50, 20, 30, 40]
```

5. extend method

```bash
    nums1 = [10, 20, 30, 40]
    nums2 = [100, 200, 300, 400]
    nums1.extend(nums2)
    print(nums1)

    [10, 20, 30, 40, 100, 200, 300, 400]
```

6. remove method

```bash
    nums = [10, 20, 30, 40]
    nums.remove(10)
    print(nums)

    [20, 30, 40]
```

7. clear method

```bash
    nums = [10, 20, 30, 40]
    nums.clear()
    print(nums)

    []
```

8. Pop

```bash
    nums = [10, 20, 30, 40]
    nums.pop()
    print(nums)

    [10, 20, 30]
```
